Sound and Vibration Analysis

The goal of this project was to create a no-code platform to help engineers analyze sound and vibration data from electric cars using various Deep Learning models.
It includes a dashboard to visualize and preprocess the data, an interface to train models such as CNNs, RNNs, and LSTMs from scratch, and a model deployment section to deploy trained models.
I took part in the implementation of the dashboard and the model training interface.

Tech stack: Python, Pytorch, FastAPI.
