Robot Arm Control

The goal of this project was to train and evaluate the OpenVLA model, a Vision Language Action model, on a specially created dataset for robotic arm.
I took part in the implementation of the model training and inference pipelines, and the development of the IsaacGym simulator used to evaluate the model's performance.
Dataset and models soon available on AIHub.

Topics : VLA models, robot, simulation

Tech stack: Python, PyTorch, Transformers, IsaacGym.
