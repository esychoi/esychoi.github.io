LLM based Translation

The goal of this project was to create a LLM-based professional translator from English to French in the healthcare domain using a specifically curated dictonary.
I took part in the prompt engineering of the model and its evaluation.

Topics : Prompt engineering, LLM evaluation, LLM based translation.
Tech stack: Python, OpenAI, NLTK, FastAPI.

AI powered School Textbook

The goal of this project was to create an AI-powered digital textbook to help Korean elementary school students learn English more effectively.
It includes a students page with LLM generated quizzes, and a teachers page to follow up on the students' progress.
I took part in the implementation of a chatbot answering questions about the digital textbook and its functionalities.

Topics : Text embedding, Vector Search.
Tech stack: Python, Transformers, NLTK, FastAPI.

Sound and Vibration Analysis

The goal of this project was to create a no-code platform to help engineers analyze sound and vibration data from electric cars using various Deep Learning models such as CNNs, RNNS, and LSTMs.
It includes a dashboard to visualize and preprocess the data, an interface to train classification and anomaly detection models from scratch, and a model deployment section to deploy trained models.
I took part in the creation of the whole platform, from the architecture design to the final deployment.

Topics : MLOps, AutoML.
Tech stack: Python, Pytorch, FastAPI, MLflow, Ray, Onnx.

Robot Arm Control

The goal of this project was to train and evaluate the OpenVLA model, a Vision Language Action model, on a specially created dataset for robotic arm.
I took part in the implementation of the model training and inference pipelines, and the development of the IsaacGym simulator used to evaluate the model's performance.
Dataset and models soon available on AIHub.

Topics : VLA models, robot, simulation.
Tech stack: Python, Pytorch, Transformers, IsaacGym.

AI Excavator

The goal of this project was to develop a suite of AI models capable of controlling an excavator to perform a range of tasks on construction sites, including digging, lifting, and Leveling.
I specifically took part in the implementation, training and evaluation of an excavator bucket trajectory generation model based on geometric features of the terrain to dig.
We used a Dynamic Movement Primitives (DMP) based system coupled with a Multi-Layer Perceptron (MLP) as the shaping force modulator to generate the trajectory of the bucket.

Topics : trajectory generation, AI based heavy equipment.
Tech stack: Python, Pytorch.
